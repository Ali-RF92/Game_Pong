# 🏓 Pong Game (Python + Pygame)

This is a simple **Pong game** built with Python and Pygame.
I created this project as part of my learning journey to better understand game development, real-time logic, and working with graphics in Python.

It’s a classic: two paddles, one ball, and fast reactions.

---

This project recreates that experience using Python.

---

## ✨ Features

* Smooth paddle movement
* Ball physics (direction, bounce)
* Collision detection (walls & paddles)
* Simple and clean game loop
* Built with modular structure (separate logic & constants)

---

## 🛠️ Built With

* Python 3.11
* Pygame

---

## ⚙️ Installation

1. Clone the project:

```bash
git clone https://github.com/your-username/pong-game.git
cd pong-game
```

2. Create virtual environment:

```bash
python -m venv venv311
venv311\Scripts\activate
```

3. Install dependencies:

```bash
pip install pygame
```

---

## ▶️ Run the Game

```bash
python game.py
```

---

| Key   | Action              |
